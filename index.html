TYPES OF CSS:-

INLINE CSS:-

Inline CSS is a way of applying styles directly to an HTML element using the style attribute. It allows you to define CSS properties for a single element without using an external stylesheet.

INTERNAL CSS:-

Internal CSS is a method of applying styles directly within an HTML document using the style tag inside the head section. It is used when you want to apply styles to a single page without affecting other pages.

EXTERNAL CSS:-

External CSS is defined in a separate .css file and linked to an HTML document using the link tag inside the head section. This method helps in keeping the HTML clean and improves maintainability.

GRADIENTS

RADIAL GRADIENT

Radial gradient:

Radial gradient is a type of gradient where colors transition outward in a circular pattern from a central point. Unlike a linear gradient, which changes color along a straight line, a radial gradient spreads colors in all directions from a central focus.

Key Characteristics of Radial Gradients:

1.The color transition moves outward from a central point.

2.The gradient can be circular or elliptical.

3.The starting color is at the center, blending outward to the ending color(s).

4.You can have multiple color stops for more complex effects.

LINEAR GRADIENT

linear Gradient:

A linear Gradient is a gradual transition between two or more colors along a straight line. It is commonly used in graphic design and web development to create smooth color blending effects.

CONIC GRADIENT

Conic Gradient

Conic Gradient A conic gradient is a type of gradient where colors transition around a central point in a circular or angular fashion, similar to a pie chart or a spinning wheel. Unlike linear or radial gradients, conic gradients change color based on the angle from the center.
